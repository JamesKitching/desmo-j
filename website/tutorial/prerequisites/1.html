Java

As DESMO-J is a Java-based framework it relies on Sun's Java technology. Therefore it is necessary for you to have an installation of the Java Development Kit (JDK), version 7 or higher at your fingertips.
You can download the latest JDK from Oracle for free.

DESMO-J

Of course, you will need DESMO-J itself. You can always obtain the latest version via the DESMO-J home page or directly from the DESMO-J download page.
Please note that DESMO-J is published under the Apache License, Version 2.0.

An Integrated Development Environment (IDE)

We strongly recommend the use of an IDE as building models with the DESMO-J framework is -- in essence -- developing Java programs. Which IDE to choose depends on personal preferences, availability, quality and price. Some commonly used IDEs for Java are listed below (in alphabetical order):

  • Eclipse
  • IntelliJ IDEA
  • JBuilder
  • JDeveloper
  • NetBeans

Please note that this list does not claim to be exhaustive; nor does exclusion of an IDE from this list mean we don't recommend its use.
