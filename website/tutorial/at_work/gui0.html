Since Version 2.0, DESMO-J offers a simple graphical user interface to facilitate experimentation with models. In the applet window below you can set model and experiment parameters before you run the model by clicking on the Start button. In the status bar at the bottom of the window you can see the progress of the experiment.

The Observers panel of the applet window displays graphical observers for relevant model variables. In our example, there are two time series, recording the length of the waiting queue (i.e. how many ships are waiting to dock at the quay) and the number of available berths.

After a simulation is finished, you can have a look at the model's results by clicking on the Reports tab. All four automatically generated output files (report, trace, debug, error) can be easily accessed and viewed.

Try it out!
