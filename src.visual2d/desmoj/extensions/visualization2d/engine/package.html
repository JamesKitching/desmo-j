Top package of 2d animation engine. It contains the following subpackages:
  • command classes for command processing
  • model describes the logical part of animated objects
  • modelGrafic are swing components and describe the grafical part of animated objects
  • orga contains some interfaces and classes used in model
  • util some utilities used in animation engine
  • viewer contains the animation viewer
  • viewer.files contains icons and helptext for viewer
The animation engine is logical independent from Desmo-J
The classes TestFrame, WriteBsp0 and WriteBspArztPraxis are only for testing purposes.
